# Description
Appblog is an App coded in React Native to create and publish posts about any topic.

## Build Setup

``` bash
# clone project
$ git clone git@github.com:marioposada/react-native-blog.git

# install dependencies
$ npm install

# serve with host at localhost:19000
$ npm start
$ yarn start
```
